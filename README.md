RETR0 is a mini programming language and virtual machine designed for learning, experimentation, and a hacker-style aesthetic. It features:

Glitchy, animated ASCII banner with colors

Watch Dogs-inspired logo

Simple syntax for printing, variables, loops, conditions, and functions

Interactive demo mode

Easy to extend for custom commands

Features

PRINT: Print messages or variables

LET: Define and assign variables

ASK: Prompt user input

REPEAT ... END: Loop blocks

IF ... ELSE ... END: Conditional execution

FUNC / CALL / RETURN: Define and execute functions

Glitchy Animated Banner: Cool hacker-style effect in terminal

ANSI Colors: Makes output visually appealing

Installation
git clone https://github.com/yourusername/retr0.git
cd retr0

Files

retr0.py: Main launcher with animation, banner, and file execution
